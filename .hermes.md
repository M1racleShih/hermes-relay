# Hermes Learning Project Context

This repository is a study/worklog repo for becoming a core developer of NousResearch/hermes-agent and designing A2A support.

When assisting here:

- Prefer source-grounded answers over broad summaries.
- Use visualization when it clarifies architecture, lifecycle, state machines, or dispatch flows. Choose Mermaid, ASCII, tables, or structured text naturally, and explain any diagram in prose.
- Keep changes small and commit-ready.
- Treat A2A as a protocol adapter around AIAgent unless evidence shows a core change is required.
- Do not expose secrets, memory, prompts, raw reasoning, or internal tool implementation details in proposed A2A AgentCards or events.
